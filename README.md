# CodeStart - Online Learning Platform

CodeStart is an online learning platform for programming, design, languages, and business courses.

## Features
- React + Tailwind CSS
- 4 pages (Home, Courses, About, Contact)
- Contact form with validation
- Audio player & YouTube videos
- Mobile responsive design
- SVG illustrations

## Technologies
- React 18.2.0
- Tailwind CSS 3.3.0
- Lucide React Icons
- JavaScript ES6+

## Live Demo
https://beishenshugyla08.github.io/my-codestart

## Installation

```bash
npm install
npm start
```
